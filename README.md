# Pro Weather

Egy modern, reszponzív időjárás-alkalmazás, amely valós idejű adatokat biztosít a világ bármely pontjáról.

## Jellemzők
*   **Valós idejű keresés:** Keresés városnév alapján.
*   **GPS lokalizáció:** Aktuális időjárás megjelenítése a saját tartózkodási helyeden.
*   **Részletes adatok:** Hőmérséklet, hőérzet, páratartalom, szélsebesség és légnyomás.
*   **Előrejelzések:** Óránkénti és 7 napos előrejelzés.
*   **Térképes nézet:** Interaktív térkép a helyszín megtekintéséhez.
*   **Vizualizáció:** Csapadék esélyét mutató diagram.

## Technológia
*   **HTML5 & CSS3:** Reszponzív, modern felület.
*   **JavaScript:** Aszinkron adatkezelés (`fetch` API).
*   **API:** [Open-Meteo](https://open-meteo.com/) időjárási adatokhoz.
*   **Térkép:** [Leaflet.js](https://leafletjs.com/) térképi megjelenítés.

## Használat
1. Klónozd a tárolót.
2. Nyisd meg az `index.html` fájlt egy modern böngészőben.
3. Keress rá egy városra, vagy használd a "Saját hely" gombot.
